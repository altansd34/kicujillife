AColor Magic Test: ItCan Read Your Mind

Publish date: 2024-08-17

We all love magic and tricks. So today we are going to create a little bit of magic together by performing some simple actions. This test is a special combination of numbers and colors. It’s not difficult! The only thing that you need to remember is the color that belongs to your number.

We at Bright Side prepared a test for you. Take a seat and have some rest from your daily routine with our magic test that can read your mind.

Pick a number between 1 and 30.

For example, we picked 15.

Take a look at the box below and see if it has the number that you picked. If yes, note the yellow color.

There is number 15 in this table, so we noted the yellow color.

Take a look at the box below and see if it has the number that you picked. If yes, note the orange color.

There is also the number 15 here. We noted the orange color.

Take a look at the box below and see if it has the number that you picked. If yes, note the purple color.

Yes! Again we found 15! The purple color is noted.

Take a look at the box below and see if it has the number that you picked. If yes, note the red color.

And, again the red color has been noted.

Take a look at the box below and see if it has the number that you picked. If yes, note the green color.

Here we didn’t find our number 15.

Now add up the number according to the colors that you noted. Is it the number that you picked?

So we had yellow — 1, orange −2, purple — 4, and red — 8. Let’s add up the numbers.

1 + 2 + 4 + 8 = 15!

That is our number that we picked at the beginning!
